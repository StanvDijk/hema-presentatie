Opdracht 1 — optimalisatie

SEO-aanpak in 3 pijlers

📝

On-page

  • Zoekwoord in H1 + title + meta
  • Unieke productbeschrijvingen
  • Alt-tags op afbeeldingen
  • Interne linkstructuur
  • Structured data (Product, FAQ)
✍

Content

  • Seizoensblogs (Back to School, Sint)
  • Inspiratiepagina’s
  • FAQ-secties per categorie
  • Video-content
  • Linkbuilding via PR
⚙

Technisch

  • Core Web Vitals (LCP, INP, CLS)
  • Mobile-first responsive
  • XML-sitemap + robots.txt
  • Canonical tags
  • HTTPS + snelle server
